# PartSense
Enclosure Particule Sensor

Enclosure particule sensor to monitor particule and dust level during a print.

Hardware:
  NodeMCU v1.0
  DHT22
  PlaTower PMS5003
  
 Software:
  Arduino extension
  ESP8266 arduino extension
  
 Note: To flash the firmware, unplug the PMS5003 from the RX pin.
